Viewing decentralized smart monitoring data from the field.

Project Overview

This web application serves as the cloud-based counterpart to the NoNet local monitoring system. It receives, stores, and visualizes sensor and anomaly data that has been forwarded from a remote, offline-first network.

Use the Dashboard to see live charts and metrics, or visit the Map to view the geographic location of sensor nodes.

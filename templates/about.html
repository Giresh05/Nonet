About The NoNet Project

ESP-NOW Protocol

The core of the field network uses an ultra-low latency, high-frequency communication protocol for decentralized node-to-node communication â€” no Wi-Fi required.

Cluster-Based Mesh

The system is designed as a self-healing, redundant node network with failover support and alpha node reassignment logic built-in for maximum uptime in the field.

Fail-Safe Logging

Each node in the field has on-device SD card logging to enable offline capture of all sensor readings, ensuring no data is lost even during communication outages.

ML-Ready Insights

The local server supports on-the-fly anomaly scoring and ensemble-based alert prioritization using isolation forest and online learning models before forwarding data to the cloud.

Local & Cloud Dashboards

The project features two distinct user interfaces: a local dashboard for in-field control and diagnostics, and this cloud-based platform for remote monitoring and data visualization.
